# NOUN Study Centre Gap Analysis

Last updated: 2026-06-19

## Current NounCompass Coverage

Current study-centre content found during this audit:

- `noun-study-centres-in-lagos`
- `/study-centres` category hub

This means the existing coverage is useful, but still narrow.

## Verified Inventory vs Current Coverage

| Area | Current Coverage | Research Reality | Gap |
| --- | --- | --- | --- |
| National centre directory | Missing | Verified and partially verified centres exist across many states | Critical gap |
| Abuja coverage | Missing | FCT is one of the strongest verified clusters | High |
| Kano coverage | Missing | Kano is the largest cluster in the audited inventory | High |
| Ogun coverage | Missing | Abeokuta and other Ogun-linked centres appear in the dataset | High |

| Benin / Edo coverage | Missing | Benin is directly verified in the official search pass | Medium |

| Special / correctional centre guidance | Missing | The inventory includes these categories, but they require careful framing | Medium |

## Missing Cities / States With Strongest Opportunity

Highest-priority missing expansion areas:

- FCT / Abuja
- Kano
- Ogun
- Enugu
- Edo / Benin
- Kaduna

## Missing Workflows

The current Lagos page is too general to fully cover these student workflows:

- how to verify a centre before travelling
- how to choose the right centre during admission
- how to confirm whether a centre contact is official
- how to interpret special or correctional centre listings
- how to move from centre discovery into registration, exams, results, and support

## Current Weakness In The Existing Lagos Page

The current Lagos article is generic and cites an official source URL that does not match a verified current centre-directory page from this audit. It is still useful as a student-preparation guide, but it does not yet function as a research-backed location authority page.

## Safe Improvement Direction

The safest future improvement path is:

1. build a national verified-centre pillar
2. add Abuja, Lagos, and Kano support pages first
3. add a centre-verification workflow page
4. expand only into cities that remain supported by verified or strongly partially verified data
