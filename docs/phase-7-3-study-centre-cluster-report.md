Editorial rules applied:

- only `VERIFIED` rows were presented as confirmed centre facts
- `PARTIALLY VERIFIED` rows were labeled clearly and treated as recheck-required
- `UNVERIFIED` rows were withheld from factual location guidance
- no new research was added in this phase
- no screenshots were created or referenced
