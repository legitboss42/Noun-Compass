# NOUN Study Centre Cluster Plan

Last updated: 2026-06-19

## Rule For This Plan

This cluster plan uses only the centres and categories supported by the Phase 7.3 research pass. It avoids publishing location-detail pages for cities that still remain weakly verified.

## Recommended Pillar Page

`Full List Of Verified NOUN Study Centres In Nigeria`

Why this should be the pillar:

- students often search for the full list first
- the current NounCompass site has only a Lagos article, not a national centre authority page
- the research uncovered 102 third-party rows but only 31 centres are currently strong enough to treat as verified
- a verification-first national list would differentiate NounCompass from recycled centre lists
