## Check which fields the portal lets you edit

To update your NOUN profile:

1. Log into portal
2. Go to **"My Profile"** (top right)
3. Click **"Edit Profile"**
4. Update information (address, phone, email, etc.)
5. Click **"Save Changes"**
6. Portal saves changes immediately

The portal decides which fields can be edited and when a change becomes active. Save the confirmation shown after submission and verify the updated value on the refreshed profile.

## What You Can Update

The profile may show contact, address, personal, study-centre, and programme details. Some may be editable while others may be locked. Treat the controls visible on your account as the current boundary.

### Step 4: Update Information

Edit fields as needed:

**Phone Number:**
- Clear old number
- Enter new number with country code (+234 for Nigeria)
- Format: +234 XXX XXX XXXX

**Email Address:**
- Clear old email
- Enter new email
- Use an address you control and follow any verification instruction the portal displays

**Address:**
- Update street address
- Update city/state
- Update postal code

**Emergency Contact:**
- Name
- Phone number
- Relationship

### Step 5: Save Changes

1. Review all changes
2. Click **"Save Changes"** or **"Update Profile"**
3. Save the confirmation or error message
4. Refresh the profile and confirm what actually changed

## Troubleshooting

### Issue: "Can't change email address"

Read the message shown after saving. If the portal asks for verification, follow that route. If the field is locked or the change does not appear, preserve the screen and contact official support.

### Issue: "Edit button not showing"

Try:
1. Refresh page
2. Clear cache
3. Try different browser
4. Contact support

### Issue: "Can't change program/student ID"

Do not try to work around a locked programme or student-ID field. Save the incorrect value and ask the current official academic or account-support route how it should be corrected.

Use only the official identity-correction process shown by NOUN. This page does not verify that a name field can be changed from the profile screen.

No, only if information changed.

Use the email field and account guidance shown by the current portal; this page does not verify support for multiple addresses.

Update the address if the field is editable, then confirm the saved value. If it is locked, request official account support.

## Requirements Before You Start

Review the existing value and gather any official record that proves a correction is needed. Never give a helper your password or one-time code to edit the profile.

## Common Mistakes

- Copying another student's field values or correction route.
- Editing several fields without noting their original values.
- Leaving before confirming which changes were saved.
- Giving a helper the password or reset code to edit the profile.
- Repeating a failed update without saving the error message.
