## Quick Answer

If your NOUN TMA score is not showing, do not check only the TMA activity page. In the observed eLearn session, the clearer score surface was the course-specific grade report inside `Grades`. First verify the correct course and TMA, then check the broader `Grades` overview, then open the course-specific grade breakdown. If the mark still is not visible, prepare safe evidence before contacting support.

## Start With A Workflow Check, Not A Panic Check

The observed session showed a common trap: the TMA activity page and the grade-report page do different jobs.

- The TMA activity page shows attempt state.
- The grade report shows how marks are recorded inside the course.

So when a score appears missing, your first question should be:

`Am I checking the right page for the right kind of answer?`

That single shift solves a lot of confusion.

## What We Actually Observed About Scores

The live eLearn session revealed two grade surfaces:

1. `Grades` overview
2. course-specific grade report

The broader overview showed course totals across enrolled courses. The course-specific grade report showed the detailed TMA line items and the overall course total.

In the observed `Numerical Analysis (MTH213_261)` course, the detailed report included:

- `Tutor Marked Assignment 1 (TMA1)`
- `Tutor Marked Assignment 2 (TMA2)`
- `Tutor Marked Assignment 3 (TMA3)`
- calculated weights

## Step-By-Step: What To Check First

### Step 1: Confirm the correct course

Make sure you are checking the grade record for the exact course connected to the TMA. Students can lose time by opening the right platform but the wrong course.

### Step 2: Confirm the exact TMA label

The observed course separated `TMA1`, `TMA2`, and `TMA3`. If you remember only "my TMA score is missing," you may end up checking the wrong line item.

### Step 3: Open the broader Grades area

Use the visible `Grades` route on eLearn. This helps you confirm whether the course total itself is already showing.

### Step 4: Open the course-specific grade report

This is the stronger diagnostic step because it can reveal whether:

- the course total is visible but a specific TMA line is unclear
- the TMA line is present with a score
- the grade record is visible but incomplete

### Step 5: Compare the grade area with the TMA activity page

Return to the TMA page and check whether the attempt summary shows the work as finished. If the TMA is not even marked `Finished`, the missing score question may actually be a submission-state question.

For help with that part of the workflow, read [How To Submit TMA On NOUN eLearn](/articles/how-to-submit-tma-on-noun-elearn).

## Use The TMA Activity Page As A Status Check

In the observed workflow, the TMA page itself was still valuable, but not because it was the clearest score location. It was valuable because it showed:

- `Status Finished`
- `Started`
- `Completed`
- `Duration`
- `Review not permitted`

Those fields help you answer an earlier question:

`Did the platform record the attempt as completed?`

If you do not know that answer, a score complaint can become messy quickly.

## What A Missing Score Can Mean In Practice

Because this guide stays inside observed facts, it does not invent backend explanations. Instead, it focuses on the real verification questions students can answer safely:

- are you checking the right course?
- are you checking the right TMA?
- are you looking at `Grades` or only at the TMA page?
- does the TMA page show the attempt as finished?
- do you have safe evidence of the current page state?

That approach is more useful than guessing a hidden cause you cannot confirm from the interface.

## A Safer Verification Routine

Use this routine before you open support:

1. open the course-specific grade report
2. look for the exact TMA line item
3. note whether the course total is visible
4. return to the TMA page
5. confirm whether the attempt is marked `Finished`
6. save safe screenshots of both surfaces
7. record the course code and TMA label
8. only then prepare support follow-up

This turns a vague complaint into a traceable one.

## Common Mistakes Students Make

- checking only the TMA page and never opening `Grades`
- checking `Grades` but not opening the specific course report
- confusing `TMA1`, `TMA2`, and `TMA3`
- assuming a missing visible score means the TMA never submitted
- escalating without a course code or safe screenshot

These are workflow mistakes, not intelligence mistakes. The observed platform simply divides information across multiple pages.

## Troubleshooting

### The course total is visible but the TMA detail feels unclear

Open the course-specific grade report and examine the TMA line items one by one. The detailed report is better than the summary view.

### The TMA page looks finished but I still do not see a mark

That is exactly when the grade-report check matters most. The TMA page confirms attempt state, but the grade report confirms mark visibility.

### I only checked the dashboard before

Return to the `Grades` area. The dashboard is useful for entry and navigation, but it is not the clearest place to diagnose a missing TMA score.

### I am not sure whether I am in the right course

Verify the course code through the course page first. If needed, use [How To Find TMA On NOUN eLearn](/articles/how-to-find-tma-on-noun-elearn) and [the wider eLearn guide](/articles/noun-elearn-and-tma-guide).

### I still need official help

Use the eLearn support route, but only after collecting safe evidence.

## What To Prepare Before Escalating A Missing Score

The support step becomes much easier when you prepare the right record first. Based on the observed workflow, your checklist should include:

- the exact course code
- the exact TMA label such as `TMA1`, `TMA2`, or `TMA3`
- a safe screenshot of the TMA page showing the attempt state
- a safe screenshot of the course-specific grade report
- the difference between what is visible and what you expected to see

This matters because a statement like `my TMA score is not showing` is too broad on its own. A more useful message is:

`I checked the course-specific grade report for MTH213_261 and the TMA1 line is not visible, while the TMA page shows a finished attempt state.`

That kind of report is grounded in observed page surfaces, not guesswork.

## Frequently Asked Questions

### Where should I look first when a TMA score seems missing?

Check the course-specific grade report first because that was the clearest score surface in the observed session.

### Is the TMA page enough by itself?

No. The TMA page is better for attempt status. The grade report is better for marks.

### What if the TMA page says Finished?

That confirms the attempt state, but you should still check `Grades` and the course report for the actual score.

### What should I save before contacting support?

Save the course code, the TMA label, a safe screenshot of the TMA state, and a safe screenshot of the grade surface you checked.

### Can NOUN Compass fix a missing TMA score?

No. NOUN Compass is an independent guide site and cannot access or change official student records.
