## What Makes A Useful NOUN Support Ticket?

Use the current official NOUN support route that matches your issue. Describe one problem clearly, include the relevant course code or transaction reference, attach only safe evidence, and keep the acknowledgement or ticket reference. Do not submit repeated reports or expose passwords and payment credentials.

## What This Guide Covers

This guide explains how to turn a confusing payment, portal, registration, examination, result, or course-material problem into a clear support request that an authorised reviewer can understand.

## Requirements Before You Start

Before writing the ticket, collect the exact course, transaction, session, date, error message, and safe screenshot connected to one problem. Remove passwords, one-time codes, card details, and unrelated identity documents.

## Step-by-Step Support Ticket Guide

1. Identify one problem only, such as a missing result, failed payment reflection, or unavailable course-registration record.
2. Confirm the exact official route or portal page connected to that problem before writing the report.
3. Collect the minimum evidence needed: course code, transaction reference, session, semester, date, safe screenshot, or prior ticket reference.
4. Write a short timeline that explains what you did, what you expected, what actually happened, and the precise help you need.
5. Submit the report through the current official NOUN support route and save the acknowledgement or ticket ID immediately.
6. If follow-up is needed, reply to the same case with new evidence instead of opening duplicate complaints.

## Common Mistakes to Avoid

- Opening several tickets for the same issue before the first one is reviewed.
- Writing a complaint without the exact course code, session, transaction reference, or error detail.
- Sharing screenshots that expose passwords, one-time codes, or unrelated private records.
- Assuming the ticket resolves itself without checking responses or deadline-sensitive next steps.

## When A Ticket Needs More Detail

### Payment did not reflect

State the intended payment, amount, date, transaction reference, current status, and the exact help needed. Attach only safe evidence and avoid making a second payment until the first status is clarified.

### A result is missing

Include the course code, session, semester, registration evidence, examination evidence where available, and the result view. Explain what appears and what you expected to appear.

### A registration course is missing

Identify the programme, level, semester, course code, and the current registration screen. Explain the checks already completed and ask for a specific clarification. If the main issue is a code mismatch, use [NOUN Missing Course Code Error](/articles/noun-missing-course-code) to organise the evidence first.

### A finance question is really about eligibility or affordability

Do not send a vague message like `I cannot pay`. State whether the issue is a current portal charge, payment reflection, NELFUND uncertainty, or a category-specific fee question. These guides help you narrow it first:

- [How to pay NOUN school fees safely](/articles/how-to-pay-noun-school-fees)
- [Can I Pay NOUN School Fees in Installments?](/articles/noun-installment-payment)
- [NELFUND Frequently Asked Questions For NOUN Students](/articles/nelfund-frequently-asked-questions-for-noun-students)

### A course material or study-resource issue is blocking progress

Identify the exact course code, material source, and broken or missing step before reporting it. Students often need one of these pages before opening a ticket:

- [NOUN Course Materials: Official PDF Downloads vs Hardcopy](/articles/noun-course-materials-pdf)
- [GST302 Study Guide](/articles/gst302-summary)

### You received no response

Keep the original reference and follow the current official escalation guidance. A concise follow-up tied to the first report is usually more traceable than several duplicate tickets.

### Your screenshot contains private information

Redact passwords, one-time codes, payment-card details, unnecessary identity numbers, and unrelated records. Share only what the authorised process requires.

### You contacted the wrong channel

Confirm the current official route for the type of issue. NOUN Compass can explain how to organise a report but cannot access accounts or resolve official student records.

## Important Warnings

- Describe one problem clearly instead of combining unrelated requests.
- Include the course code, payment reference, or result detail needed to identify the issue.
- Remove unrelated personal information from any screenshot you attach.
- Keep the first ticket number and use it when following up.
- Do not open repeated tickets unless the current support route tells you to.
