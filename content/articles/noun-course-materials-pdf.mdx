## Where to get NOUN course materials

NOUN offers course materials in two formats:
- **PDF (Digital):** Free, instant access, download from portal
- **Hardcopy or printed options:** availability and cost must be confirmed through current official instructions

**Which to choose?**
- If you have internet: Get PDF (save money)
- If you prefer printed: Order hardcopy (costs money but physical)
- Many students get PDF but print themselves at cybercafe

## Where To Look For A Course Material

This guide explains where to look for official course materials, how to use PDFs responsibly, when hardcopy may help, and how to avoid unsafe download habits.


## PDF Course Materials: Download & Access

### How to Download PDF Materials

1. Log into NOUN portal
2. Go to **"Courses"** → Select course
3. Click **"Course Materials"** or **"Study Materials"**
4. Click **"Download PDF"** or **"View Online"**
5. PDF opens in new tab or downloads
6. Save to your computer

### PDF Advantages

- ✓ Free
- ✓ Instant access
- ✓ Can search text
- ✓ Accessible on phone/tablet/computer
- ✓ No shipping delay

### PDF Disadvantages

- ✗ Requires internet to access (unless downloaded)
- ✗ Eye strain from screens for long study sessions
- ✗ Battery drain on devices
- ✗ Easy to lose if computer crashes

## Hardcopy Course Materials: Ordering

### What Hardcopy Questions Require Extra Care

The official NOUN eCourseware source used by this site strongly verifies the PDF side of the workflow. Hardcopy handling is less directly documented in the indexed official source set, so students should treat printed-material expectations more cautiously and confirm the current route through their own official record or study-centre instruction.

### Hardcopy Advantages

- ✓ Physical book you can hold and annotate
- ✓ No internet required to study
- ✓ No eye strain
- ✓ Easy to refer back to sections
- ✓ Professional quality printing

### Hardcopy Disadvantages

- ✗ Costs extra money
- ✗ 2-4 week delivery delay
- ✗ Bulky to carry around
- ✗ Can't search text quickly
- ✗ Hard to revise (can't edit notes easily)

## Requirements Before You Start

Use the exact course code and title from your registration. Download only from a source you can verify, and keep portal credentials out of third-party material sites.

## Step-by-Step Verification Process

1. Copy the exact course code and title from your registration.
2. Search the official material library by that code.
3. Check that the file title matches the registered course.
4. Save the official PDF locally when a download is available.
5. Report a missing or mismatched material without using an unsafe mirror.

## Common Mistakes

- Downloading a file because its title looks similar to your course.
- Using a material without matching the exact code and version.
- Closing the official library before saving a working copy or source link.
- Entering portal credentials on an unofficial download site.
- Repeating failed downloads without checking whether the file is actually available.

## Troubleshooting

If the expected material is absent, check the exact code and title once more. Report the registered course, library search result, and missing file through the current official route instead of downloading an unverified copy.

## Official Sources and Verification Notes

An available file may belong to another course version or semester. Match the code and title with your registration and use the current official material as the learning source.
