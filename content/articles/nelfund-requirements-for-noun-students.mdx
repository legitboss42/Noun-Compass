## What Details Does The Public Form Request?

The requirements we can safely state from the verified public NELFUND flow are:

- institution selection
- matric number
- JAMB registration number
- date of birth

Official NOUN material confirms participation and approved disbursement for some students. The public form confirms several input checks, but the reviewed sources did not provide one complete later-stage document checklist for every applicant.

## Requirements Visible In The Public Flow

This article focuses on one practical question: what does the verified NELFUND process currently expect from a NOUN student at the public verification stage?

It explains:

- the inputs that were directly verified
- how those inputs fit into the application sequence
- what students should prepare before starting
- which `requirements` claims remain unverified

If you need the broad participation question first, start with [Is NOUN Eligible For NELFUND?](/articles/is-noun-eligible-for-nelfund).

## The Safest Way To Think About Requirements

A requirement is not the same as a rumor, and it is not the same as a highly plausible guess.

For this cluster, the research standard is strict:

- if the official public flow showed it directly, it can be described as verified
- if the official NOUN announcement confirmed it, it can be treated as verified
- if students ask about it often but the reviewed official sources did not settle it, it must stay unverified or partially verified

That rule matters most on student-finance topics because people make real decisions from small wording differences.

## Verified Requirement 1: Institution Selection

The public NELFUND registration flow explicitly showed an educational verification step with `Select Institution`.

That is a clear signal that the application checks school-linked data near the start of the process.

For NOUN students, the finance application is tied to school identity rather than being only an expression-of-interest form.

## Verified Requirement 2: Matric Number

The same verified educational-verification step also showed `Matric Number`.

This is one of the most important NELFUND requirements for NOUN students because it is the point where the system begins linking the application to a live student record.

In practical terms, it means a student should not start carelessly with:

- an old or uncertain matric record
- copied details from another student
- assumptions about what the school record contains

This is also why record consistency matters across other workflows such as [How To Register NOUN Courses](/articles/how-to-register-noun-courses), [NOUN Support Ticket Guide](/articles/noun-support-ticket-guide), and [Update Profile On NOUN Portal](/articles/update-profile-nouonline).

## Verified Requirement 3: JAMB Registration Number

The public NELFUND flow also showed a JAMB-based verification stage requiring a JAMB registration number.

This is not a rumor. It is a visible public step in the current application flow.

That does not automatically answer every regularization question, but it does settle one thing clearly: JAMB-linked verification is currently part of the NELFUND process.

## Verified Requirement 4: Date Of Birth

The JAMB-based step also required date of birth.

That matters because students sometimes treat only the JAMB number as important, then overlook the fact that identity-linked verification usually depends on more than one field matching cleanly.

## What The Verified Requirements Suggest

Taken together, the verified inputs suggest a pattern:

1. NELFUND wants to tie the applicant to a real student record
2. it wants that record to line up with school-linked information
3. it also wants JAMB-linked identity or student-status verification

NELFUND questions overlap with admission, registration, fees, and support because school-record mismatches can interrupt the finance application.

## What Is Not Verified As A Requirement Yet

This is where the article needs discipline.

### Postgraduate status as a settled requirement rule

The reviewed official sources did not clearly say whether postgraduate NOUN students are included or excluded. So the site should not present postgraduate eligibility as a settled requirement rule.

### Returning-student or new-student timing rules

The reviewed sources did not state exactly when every new or returning NOUN student can move through the full process. Timing-based requirement claims therefore remain qualified.

### Universal JAMB regularization requirement

This is the biggest caution point. The public NELFUND flow clearly requires JAMB-linked verification. But the reviewed official sources did not fully verify that every NOUN student must complete the same JAMB regularization path before applying.

So the responsible wording is:

- `JAMB-linked verification is verified`
- `exact universal JAMB regularization requirements for all NOUN students are not fully verified`

### A complete post-email-verification document list

The reviewed sources did not provide a full later-stage document checklist, so this page stops at the requirements that were actually visible.

## A Practical Requirements Checklist Students Can Use Safely

Without crossing into assumption, students can still prepare a useful checklist before applying:

### Confirmed from the public flow

- official NELFUND portal access
- institution selection
- matric number
- JAMB registration number
- date of birth

### Sensible supporting preparation

- your own current NOUN record
- a secure place to note any mismatch or warning
- caution around identity consistency across school and JAMB-linked data

### Not safe to promise as a universal checklist item yet

- a fully verified regularization-number rule for all NOUN students
- a verified later-stage document set for every NOUN applicant
- a guaranteed timeline from application to approval

## Common Requirement Mistakes

- assuming `requirement` means `guaranteed approval`
- adding unofficial checklist items because they sound likely
- confusing verified JAMB verification with a fully verified NOUN-wide regularization rule
- starting the process without checking your own record consistency
- relying on a second-hand explanation instead of the official public flow

## Troubleshooting Requirement Confusion

### You expected only a matric number, not JAMB details

The verified public flow shows both educational verification and JAMB-based verification. So the surprise is understandable, but the JAMB step itself is real.

### You are searching for a perfect universal document list

The research did not verify a full post-email-verification checklist, so the article should not pretend to have one. Stay within the confirmed public inputs.

### Someone told you regularization is the only thing that matters

That is too narrow. The verified flow also checks institution selection, matric number, and date of birth.

### Someone told you requirements are identical for every NOUN student category

That is not supported by the reviewed evidence. Category-specific questions still need stronger official confirmation.
