## Compare the code with your curriculum

"Missing course code" error happens when:
- Course not offered this semester
- You're not eligible (prerequisite not met)
- Course not available for your program
- Portal has temporary glitch
- Wrong level for your year of study

**First check:** Compare the missing code with your current curriculum, result statement, level, and semester. Save the portal view before asking the official academic-support route which code applies.

---


## What To Check Before Choosing Another Course

This guide explains why course codes may not appear, what records to compare, what not to click, and how to report the problem clearly.


## Why Course Code Shows as Missing

### Reason 1: Course Not Offered This Semester

A course may not be available in the semester or curriculum currently attached to your record.

**What to do:** Check the current schedule and curriculum. Do not choose an alternative until the replacement is officially confirmed.

### Reason 2: Prerequisite Not Met

The course may have a prerequisite or sequence rule, but a missing code does not prove that this is the cause.

**What to do:** Compare the prerequisite guidance with your result statement and request clarification if they conflict.

### Reason 3: Not Eligible for Program Level

The code may belong to another programme level or curriculum version.

**What to do:** Verify the programme, level, semester, and curriculum version attached to the course.

### Reason 4: Wrong Course Code

You typed wrong course code.

**Solution:** Double-check course code from course schedule.

### Reason 5: Portal Glitch

The page may also be incomplete or temporarily unavailable. Refresh once and save the repeated state before reporting it.

## Step-by-Step Fix

1. **Verify course code** - Check against official course schedule
2. **Confirm course is offered** - Check semester schedule
3. **Check prerequisites** - Verify you've met requirements
4. **Reload once** - Confirm the same account and session are still active
5. **Save the missing view** - Include the course code and selected semester
6. **Contact official support** - Ask which current course or code applies

## Frequently Asked Questions

### Q1: Can I take an alternative course instead?

Only after an authorised NOUN source confirms the alternative for your programme and curriculum.

### Q2: Will missing this course delay my graduation?

It may affect progress if the course is required and remains unresolved. Confirm its status before planning later semesters or graduation.

### Q3: Can I appeal missing course to take it anyway?

This page does not verify a universal appeal route or approval rate. Ask the current official support channel what review process applies.

## Important Warnings

⚠️ **Do NOT:**
- Ignore "Missing Course Code" errors (they prevent registration)
- Assume error will auto-correct (it won't—you must fix it)
- Register alternative courses without confirming which is correct
- Wait until near graduation to address this (fix immediately)

⚠️ **Critical:**
- This error can **delay graduation if left unresolved**
- Only your department can authorize the correct course code
- Portal cannot auto-fix code mismatches

✓ **DO:**
- Report missing codes to department **immediately**
- Get written confirmation from department of correct code
- Request override from department if code cannot be added
- Follow up within 48 hours (don't let it sit)

---

## Do not substitute another course without guidance

Missing course codes need evidence, not guesswork.

Check whether you selected the correct level and semester, whether the code still belongs to the current curriculum, and whether the course is unresolved on your result statement. Keep the portal screenshot and the official reply together so you can explain any later registration mismatch.

## Requirements Before You Start

Keep the exact missing code, curriculum version, level, semester, result statement, and portal screenshot together. Do not share credentials with anyone offering an override.

## Common Mistakes

- Copying a replacement code from another student's portal.
- Selecting an alternative before checking the curriculum and result status.
- Leaving the page without capturing the missing code and selected semester.
- Giving credentials to someone promising to add the course.
- Submitting the same selection repeatedly without an official correction.

## Troubleshooting

If the code remains absent, save the selected session, programme, level, curriculum entry, result status, and missing portal view. Ask official academic support which current code or course should appear.

## Official Sources and Verification Notes

Only an authorised NOUN response can confirm a replacement, prerequisite, or correction. This page helps you document the mismatch without guessing the answer.
