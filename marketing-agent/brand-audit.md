# NounCompass Social Brand Audit

Date: 2026-07-05

## Scope

- Facebook Page
- Instagram Professional Account
- Pinterest Business Account

## Executive summary

No significant branding changes were made. The controlled browser sessions now show live, editable access for Facebook, Instagram, and Pinterest, so the current step is still an audit and recommendation pass rather than a live brand update.

## Audit findings

### Facebook

- Current state: Live Page URL confirmed and the manage-page view is available.
- Existing info: page name, Contact Us button, follower count, linked website, and page category are visible.
- Missing info: cover image, brand-safe profile image, About section cleanup, Services, featured section, and consistent website/contact details.
- Recommendation: replace the non-brand website link, standardize the About copy, and use a circular-safe logo or icon.

### Instagram

- Current state: Live profile and edit-profile screen confirmed.
- Existing info: name and username are set; the profile can be edited.
- Missing info: bio, category, highlights, visible website link, and a clearer brand-safe profile image.
- Recommendation: add a compact bio, set the professional category, and create highlight covers for Admission, Fees, TMA, Exams, Guides, Resources, and FAQs.

### Pinterest

- Current state: Live profile URL confirmed and a profile shell is visible.
- Existing info: profile exists, boards tab is available, and a Create board action is visible.
- Missing info: business profile description, featured boards, verified website, and a stronger profile image if needed.
- Recommendation: turn the profile into a structured evergreen resource hub with boards for admission, resources, study tips, exams, and website tutorials.

## Recommended brand direction

### Colour palette

- Primary ink: #12243B
- Brand green: #0E7A3E
- Deep green: #085A2D
- Soft green: #E6F5EC
- Accent gold: #D6A21D
- Background: #FFFFFF and #F7F8FA

### Typography

- Headlines: Poppins or a similar clean display sans
- Body: Inter or a similar readable sans
- Keep capitalization simple and avoid decorative fonts

## Bio recommendation

Suggested platform bio:

NounCompass helps NOUN students with admission, registration, school fees, course materials, TMA, exams, portal tutorials, and student success resources. Independent student-help platform, not the official NOUN website.

## CTA button recommendations

- Facebook: Contact Us or Send Message
- Instagram: Contact or Email
- Pinterest: Visit website / Learn more
